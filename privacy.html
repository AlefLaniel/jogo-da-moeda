Política de Privacidade do Jogo da Moeda

1. Coleta de Dados

1.1. Dados Pessoais Fornecidos pelo Usuário
Coletamos informações fornecidas pelos usuários durante o uso do aplicativo, como perguntas feitas aos espíritos. 1.2. Dados Pessoais Coletados Automaticamente
Podemos coletar dados de uso, como informações sobre as perguntas feitas e respostas recebidas.

2. Consentimento

Ao usar o aplicativo, o usuário concorda com a coleta e o tratamento dos dados conforme descrito nesta Política de Privacidade.

3. Direitos do Usuário

O usuário tem o direito de acessar, corrigir ou excluir seus dados pessoais.

4. Armazenamento de Dados

Os dados serão armazenados pelo tempo necessário para cumprir os objetivos descritos nesta política.

5. Segurança

Implementamos medidas de segurança para proteger os dados dos usuários.

6. Compartilhamento de Dados

Não compartilhamos dados com terceiros sem o consentimento do usuário.

7. Cookies

Não utilizamos cookies neste aplicativo.

8. Alterações na Política de Privacidade

Reservamo-nos o direito de alterar esta Política de Privacidade a qualquer momento.

9. Contato

Para dúvidas ou solicitações relacionadas à privacidade, entre em contato conosco.
